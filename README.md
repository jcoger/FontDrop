# FontDrop
![image](assets/TypeExplorer.png)
![image](assets/Columnview.png)

A native Mac app for exploring logo and brand typography.
Drop your logo, preview every font on your system, find the one.

**Status:** v0.1.0-beta — unsigned pre-release

## Install
1. Download `FontDrop_[version]_aarch64.dmg` from [Releases](../../releases)
2. Open the .dmg and drag FontDrop to Applications
3. First launch: right-click → Open (required for unsigned apps on macOS)

## Built with
- Tauri v2 + Rust
- React + TypeScript + Tailwind
- Framer Motion
